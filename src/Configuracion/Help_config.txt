/start -> Este comando tiene el proposito de saber si el bot esta operativo o no.
/help -> Comando usado para mostrar el menú de ayuda al usuario, el menú actual.
/reboot contraseña -> Este comando tiene el proposito de reinicial el dispositivo donde se encuentra alojado nuesto bot.
/acciones -> Este comando mostrara las acciones que vamos a comprobar.
/accionesAdd Acion1 Acion2 -> Este comando añadirá acciones a la lista.
/accionesDel Acion1 Acion2 -> Este comando borrará acciones de la lista.
/sendAciones -> Este comandopermite mandar un documento .txt con los precios de las acciones.
